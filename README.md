# Portfolio Website

フロントエンドエンジニアとしての技術力を示すポートフォリオサイト

## 技術スタック

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript (strict mode)
- **Styling**: Tailwind CSS v4
- **Content**: MDX
- **Animation**: Framer Motion
- **i18n**: next-intl
- **Theme**: next-themes
- **Testing**: Vitest + Playwright

## セットアップ

```bash
pnpm install
```

## 開発

```bash
pnpm dev
```

## テスト

```bash
# 単体テスト
pnpm test

# E2Eテスト
pnpm test:e2e
```

## ビルド

```bash
pnpm build
```
